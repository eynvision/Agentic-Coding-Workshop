# GitHub Actions Workflows

These workflow files are designed to be placed in `.github/workflows/` at the root of the repository.

They are stored in this `workflows/` folder instead as a security measure. Since this is a public repository, keeping workflow files outside `.github/workflows/` prevents anyone from the public from accidentally or maliciously triggering CI/CD actions on this repo.

To use them, copy the desired file into `.github/workflows/` when needed.
